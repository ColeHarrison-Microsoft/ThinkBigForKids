# `source-materials/`

Original handout and slide-deck materials from a prior robotics program that used the same Arduino car kits. Kept here as **reference inputs** — not student-facing during the field trip.

## What's here

| File | What it is |
|---|---|
| `Robotics Class Instructions.docx` | Word document handout that accompanied the original five-lesson `.ino` series (see [`src/reference/arduino-car-code-3/`](../../src/reference/arduino-car-code-3/)). |
| `Robotics Class Slides.pptx` | PowerPoint slide deck for the same program. |

## Privacy

The repo owner removed all PII (student names, teacher names, photos, specific school identifiers, etc.) before these files were committed. See [`AGENTS.md`](../../AGENTS.md) for the repo-wide privacy rules.

If a future revision of these files is added, **re-run a PII review pass before committing** — the original archive contained material that needed scrubbing.

## How they're used

Reference only. New student-facing handouts for the field trip should be **derived** from these (paraphrased into Markdown under `docs/lessons/` when that work begins) rather than copied verbatim. See [`src/ToDo.md`](../../src/ToDo.md) for the lesson-port task list.
